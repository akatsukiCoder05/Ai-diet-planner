## 📋 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | `/api/health` | Server health check |
| POST | `/api/generate-pdf` | Generate PDF from diet plan |
| POST | `/api/save-plan` | Save diet plan to history |
| GET | `/api/history` | Get all saved plans |
| GET | `/api/history/:id` | Get specific plan by ID |
| DELETE | `/api/history/:id` | Delete plan by ID |

## 🔧 Technical Details

### Backend Stack
- **Framework**: Express.js
- **PDF Library**: pdf-lib (pure JavaScript, no dependencies)
- **CORS**: Enabled for localhost:5173
- **Storage**: In-memory (upgradeable to database)

### PDF Generation Process
1. Client sends diet plan text + user info to `/api/generate-pdf`
2. Server creates new PDF document
3. Embeds fonts (Helvetica Bold, Helvetica, Courier)
4. Adds title, user info, and diet plan content
5. Formats text with proper wrapping and pagination
6. Returns PDF as binary buffer
7. Client downloads as file

### Data Flow
```
React App (Frontend)
    ↓ fetch()
Express Server (Backend)
    ↓ pdf-lib
PDF Document
    ↓ buffer
Client Download
```

## 🐛 Troubleshooting

### PDF Not Downloading
**Problem:** Click download but nothing happens

**Solution:**
1. Check server console for errors
2. Verify server is running: `curl http://localhost:5000/api/health`
3. Check browser console for network errors
4. Ensure dietPlan is not empty

### CORS Error
**Problem:** "Access blocked by CORS policy"

**Solution:**
- Server has CORS enabled by default
- Make sure backend URL is http://localhost:5000 (not https)
- Check server is actually running

### Port Already in Use
**Problem:** "EADDRINUSE :::5000"

**Solution:**
```bash
# Windows
netstat -ano | findstr :5000
taskkill /PID <PID> /F

# Mac/Linux
lsof -ti:5000 | xargs kill -9
```

### Dependencies Missing
**Problem:** "Cannot find module"

**Solution:**
```bash
npm install express cors pdf-lib concurrently
```
